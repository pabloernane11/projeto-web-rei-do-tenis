# Rei do Tênis - E-commerce

> Front-end de um e-commerce de tênis desenvolvido para a disciplina de Programação Web. Foco em HTML5, CSS3 e layout responsivo com Flexbox.

**🔗 Repositório Oficial:** [Acessar no GitHub](https://github.com/pabloernane11/projeto-web-rei-do-tenis)

---

## Objetivo do Sistema
O sistema Rei do Tênis tem como objetivo simular uma loja virtual de venda de tênis, proporcionando uma experiência semelhante a um e-commerce.

A aplicação permite:
* Exibir produtos (tênis) de forma organizada;
* Apresentar informações como nome, preço e imagens;
* Simular a navegação de um usuário em uma loja online;
* Trabalhar conceitos de desenvolvimento web (estrutura, estilo e navegabilidade).

Este projeto foi desenvolvido com foco educacional, visando o aprendizado de desenvolvimento web.

## Tecnologias Utilizadas
As seguintes tecnologias e ferramentas foram utilizadas no desenvolvimento do projeto:
* **HTML5:** Estruturação das páginas.
* **CSS3:** Estilização e layout (responsividade e design).
* **VS Code:** Editor de código.

## Instruções para Execução
Siga os passos abaixo para rodar o projeto na sua máquina:

1. **Baixar o projeto:** Faça o download ou clone este repositório. Extraia o arquivo `.zip` em uma pasta de sua preferência, mantendo a estrutura de pastas intacta.
2. **Abrir o projeto:** Abra a pasta extraída no VS Code (ou em outro editor de sua escolha).
3. **Executar o sistema:** Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador. Não é necessária a instalação de um servidor local.
